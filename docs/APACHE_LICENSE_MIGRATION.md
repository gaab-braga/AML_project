## What Apache 2.0 Means

### Permissions (What Others CAN Do)
 **Commercial Use**: Deploy in production systems  
 **Modification**: Improve and adapt the code  
 **Distribution**: Share original or modified versions  
 **Patent Use**: Protection from patent litigation  
 **Private Use**: Use internally without sharing changes  

### Conditions (What They MUST Do)
 **Include License**: Apache 2.0 license text required  
 **Include Copyright**: "Copyright 2025 Gabriel Braga" required  
 **State Changes**: Document modifications made  
 **Include NOTICE**: Credit your original contributions  

### Limitations (What They CANNOT Do)
 **Remove Attribution**: Your copyright stays forever  
 **Use Trademarks**: Cannot use your name for endorsement  
 **Hold You Liable**: Cannot sue you if something breaks  
 **Patent Attacks**: Cannot sue for patents then use code
